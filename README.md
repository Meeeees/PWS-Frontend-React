# PWS Gezichtsherkenning Frontend programma
### Dit is de frontend repositorie van het PWS van Mees en Stijn over gezichtherkenning, bekijk de backend repositorie [hier](https://github.com/Meeeees/PWS-Backend)

### Starten van het programma
- Als je het programma wilt starten moet je het commando `npm start` uitvoeren
- Zorg ervoor dat in `src/config.json` de variabele `Backend` gelijk is aan het ip-adres en het bijbehorende pooort van de FastAPI server op de raspberryPi

## functies
- De Website heeft de volgende functies
-- Het bekijken van de live-feed van de Raspberry PI
-- Er kan een verzoek naar de server gestuurd worden om gezichtsherkenning uit te voeren
-- Er is de mogelijkheid om een pop-up te open waar gemakkelijk gebruikers kunnen worden toegevoegd, maar in deze versie van de server is dit nog niet werkend.
